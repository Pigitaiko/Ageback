# Ageback — Cashback for the Agentic Economy

---

## Why Cashback?

**The problem**: AI agents pay billions in API fees with zero loyalty incentives.
There's no reason for an agent to prefer one provider over another.

**The solution**: On-chain cashback — agents earn rewards on every API call,
providers attract and retain traffic.

---

## How it works

```
Agent pays USDC ──► Provider API ──► Cashback allocated on-chain
     (x402)          (Claude)         (automatic, no claim needed)
```

---

## Benefits

### For Agents
- **Automatic cashback** — no integration, no claims, no extra steps
- **Just change the URL** — works with any x402-enabled agent
- **Loyalty tiers** — Bronze → Platinum, up to 1.5x multiplier
- **Referral bonuses** — earn more by referring other agents
- **On-chain & verifiable** — all rewards tracked transparently

### For Providers
- **Agent acquisition** — cashback attracts traffic over competitors
- **Sticky retention** — loyalty tiers incentivize repeat usage
- **Viral growth** — referral system drives organic adoption
- **Full control** — set your own rebate rate, deposit/withdraw anytime
- **No middleman** — direct on-chain settlement on Taiko

---

## The Stack

**x402** — HTTP-native micropayments (pay per API call)
**Taiko** — Low-cost EVM L2 (fast, cheap, Ethereum-secured)
**ERC-8004** — Agent identity standard (reputation, trust scores)
**EIP-3009** — Gasless USDC transfers (agents don't pay gas to pay)
